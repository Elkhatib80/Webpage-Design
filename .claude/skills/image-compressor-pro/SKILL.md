## Core Offering
Compress images to reduce file size while maintaining quality, supporting JPG, PNG, WebP, and GIF formats with bulk compression capabilities.

## Pricing
The service costs 0.001 USDT per compression call.

## Typical Usage Examples
Users can request compression through prompts like "Compress this image https://example.com/photo.jpg" or "Optimize this PNG."

## Expected Output
Responses include metrics showing the original file location, original and compressed sizes (e.g., 2.5MB reduced to 500KB), the percentage reduction achieved, and a URL for the optimized result.

## Technical Setup
Integration requires setting an API Key through environment variables or local configuration, with pricing calculated per invocation at the stated USDT rate.
